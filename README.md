# Fredbear's Family Diner — Static Pages

This workspace contains four HTML pages and a CSS file that implement the requested static layout (no JavaScript).

Files added:

- `index.html` — Main homepage with hero, nav pill, logo, content container, image grid, ratings, and footer.
- `news.html` — News article page with date, title, author, photo card, and article body.
- `about.html` — About page with navigation, two-column section, interests images, and footer-line.
- `apply.html` — Employment application form styled with underline fields.
- `css/styles.css` — Central stylesheet for layouts and visuals.
- `assets/` — Simple SVG placeholders for images and icons.

How to view:

Open the files in a browser (e.g.)

```bash
xdg-open index.html
xdg-open news.html
xdg-open about.html
xdg-open apply.html
```

Notes:
- The images supplied are simple SVG placeholders. Replace with real photos as needed.
- All interactions are pure HTML/CSS (no JavaScript).
